Для меня культура какого либо места это в главнюю часть народ. А народ: его повадки, поведение и оссобенности главное в культуре. В Витебске хорошо отражает народ, фестиваль Славянский базар.
Для его отражения я выбрал синею цветовую гамму ведь она находится на логотипах славянского базара и одного цвета с васильком (символом беларуси).
Первым у меня в проэкте идёт навигационная панель которая похожа на навигационную панель в макете. В следуюющей секции есть логотип славянского базара и главная информация о нём. Также там есть кнопка которая переводит на сайт фестиваля для лучшего ознокомления.
Дальше в секции идёт галерея мероприятий по годам первая кнока на неё и ведёт. Дальше на экране есть мероприятия за 2025 год, а над ними кнопки нажимая на которые у вас на экране появляется эти года. При нажатии на определённый год год который на экране пропадает, а через 50 миллисекунд поевляется тот который вы выбрали появится, но из-за transition: opacity 0.7s ease этого процесса не видно.
Дальше идёт информация про отборы на фестиваль с логотипом фестиваля. С кнопкой на сайт про отборы.
Потом идёт по макету про команду разработчиков что и реализованно у меня. А в footer сделанна просто информация но принажатии на кнопки с мероприятеями с помощью якорей перекидывает на мероприятия на этот год.
Также там по кнопкам запускаются формы для обратной связи и видео про славянский базар
